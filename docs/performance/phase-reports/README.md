# Phase Reports

Create one report per week using:
- `week-1-YYYY-MM-DD.md`
- `week-2-YYYY-MM-DD.md`
- `week-3-YYYY-MM-DD.md`
- `week-4-YYYY-MM-DD.md`
- `week-5-YYYY-MM-DD.md`
- `week-6-YYYY-MM-DD.md`

Each report should include:
1. Features shipped
2. Benchmarks executed
3. Before/after metrics
4. Regressions and fixes
5. Rollout decision
